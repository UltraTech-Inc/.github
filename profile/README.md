# UltraTech — consolidated vision

**O que fazemos**  
Soluções de tecnologia empresarial que consolidam dados e processos para fornecer insights acionáveis e previsíveis.

**Missão**  
Permitir que organizações tomem decisões melhores e mais rápidas, através de plataformas integradas, confiáveis e escaláveis.

**O que encontrará aqui**  
- Repositórios de integração (ETL, pipelines, adaptadores)  
- Ferramentas de orquestração e automação  
- SDKs e bibliotecas de integração (clientes e helpers)  
- Exemplos, templates de infra e infra-as-code (IaC)  
- Whitepapers / estudos de caso e guias técnicos

**Governança de repositórios**  
- `infra-` — infra-templates, Terraform, CloudFormation  
- `sdk-` — SDKs oficiais (ex.: `sdk-python`, `sdk-js`)  
- `connector-` — adaptadores para fontes/destinos (ex.: `connector-salesforce`)  
- `platform-` — componentes de plataforma (ex.: `platform-core`, `platform-auth`)  
- `demo-` — demos, exemplos e POCs

**Contribuição**  
1. Fork → branch com `feature/xyz` → pull request.  
2. Siga o `CONTRIBUTING.md` do repositório (linters, testes e CI obrigatórios).  
3. Issues bem descritas com steps para reproduzir e objetivo esperado.

**Licença**  
Repositórios públicos usam [MIT/Apache-2.0] (verifique cada repo).  
Projetos internos/privados podem ter política diferente (ver `LICENSE` do repositório).
